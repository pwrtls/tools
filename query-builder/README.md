# Query Builder for Power Tools

Query Builder is a powerful tool designed to simplify the process of creating and managing queries for your projects. Inspired by [FetchXML Builder](https://github.com/rappen/FetchXMLBuilder), this tool has been reimagined using React to provide a more interactive and user-friendly experience.

## Features

- **Interactive UI**: A user-friendly interface that allows users to build queries with ease.
- **Compatibility**: Built using React, ensuring smooth integration with modern web applications.
- **Reference Structure**: Uses the `solutions-viewer` directory as a reference for a structured React setup.

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/pwrtls/tools.git
   ```

2. Navigate to the `query-builder` directory:
   ```bash
   cd tools/query-builder
   ```

3. Install the required dependencies:
   ```bash
   npm install
   ```

### Usage

1. Start the development server:
   ```bash
   npm start
   ```

2. Open your browser and navigate to `http://localhost:3000` to access the Query Builder tool.
